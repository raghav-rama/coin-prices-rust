# coin-prices-rust
CLI app written in Rust🦀 to display prices of some popular coins

# directions
- clone repo and use `cargo run`
- or go to [replit](https://replit.com/@hackerb0y/DisplayRealTimeCoinPrices) and run in browser!
- cheers 🍻
